# 📈 Sales & Demand Forecasting System

## 📌 Project Overview
This project builds a machine learning model to forecast future sales demand using historical store-item sales data. The goal is to help businesses optimize inventory management and predict future demand trends.

## 📊 Dataset
Dataset used: Store Item Demand Forecasting (Kaggle)

Files:
- train.csv – historical sales data
- test.csv – future dates to predict
- sample_submission.csv – submission format

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## 🚀 Steps in the Project
1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Demand Prediction

## 📈 Model Used
Random Forest Regressor

## 📌 Result
The model predicts future sales demand based on historical data patterns.
